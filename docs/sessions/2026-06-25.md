## 今天遇到的问题

### 1. sshpass 无法在 Windows Git Bash 中传递 SSH 密码
- **症状**：`sshpass -p 123456 ssh ...` 报错 `read_passphrase: can't open /dev/tty`
- **原因**：Windows 下没有 /dev/tty 设备
- **解决**：使用 SSH_ASKPASS 方式，创建 askpass 脚本输出密码，配合 `-T`（no TTY）参数

### 2. YOLOv4-tiny 输出 NaN
- **症状**：模型推理结果全部为 NaN
- **原因**：OpenCV 4.1.1 DNN 模块的 Darknet 解析器在 ARM64 上有兼容性问题

### 3. YOLOv3-tiny 类别分数全零
- **症状**：推理输出中 box 坐标和 objectness 正常，但所有 80 个类别分数精确为 0
- **原因**：权重文件与 OpenCV 4.1.1 的 Darknet 导入器之间存在兼容性问题（推测是特定卷积层权重未被正确加载）

### 4. TensorFlow SSD MobileNet 内存崩溃
- **症状**：`std::bad_alloc` 崩溃
- **原因**：OpenCV 的 TensorFlow 冻结图解析器在 Jetson Nano 上内存分配失败

### 5. 模型下载源不可用
- **症状**：多个模型的下载 URL 返回 404
- **解决**：使用 pjreddie.com 官方源（需设置 User-Agent header），以及 TensorFlow 官方模型库

### 6. 摄像头被进程占用
- **症状**：测试脚本无法打开摄像头
- **解决**：先 kill 已有 stream_server 进程
