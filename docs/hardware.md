## 行走机构

| 参数 | 数值 |
|------|------|
| 类型 | 履带式（V形自洁纹路） |
| 宽度 | 250mm |
| 接地长度 | 750mm |
| 悬挂 | 扭杆独立悬挂 |
| 材质 | 橡胶 + 凯夫拉帘线（防盐雾） |

**振动减阻模块（仅履带方案）：**

- 干沙区：50–80Hz，振幅1–2mm，阻力↓30–40%
- 湿沙区：自动关闭

> 🧪 **备选方案：** 轻量化低压轮胎（适用于硬质沙滩，成本更低）

## 电源系统

```
电池 (4S LiPo, 5000mAh)
   │
   ├─> 主控 (5V/3A) ──> Jetson + 摄像头 + 雷达
   │
   ├─> 电机驱动 (12V/10A) ──> 行走电机 ×2
   │
   ├─> 振动模块 (12V/5A) ──> 振动电机
   │
   └─> 抓取机构 (5V/2A) ──> 舵机/气泵
```

### 续航与充电

| 模式 | 功率 | 续航 |
|------|------|------|
| 干沙（无振动） | ~30W | 120min |
| 干沙（振动开启） | ~80W | 45min |
| 湿沙 | ~40W | 90min |
| 待机/回航 | ~10W | — |

**自动回充逻辑：**

1. 电量 < 25% → 中断清扫
2. 计算回航最短路径（A\*）
3. 红外引导对接充电站
4. 电量 > 90% → 返回中断点续扫

## 环境防护

| 威胁 | 防护方案 | 状态 |
|------|----------|------|
| 海水侵蚀 | 密封舱体 + 航空铝/316不锈钢 | 🚧 测试中 |
| 沙尘侵入 | 迷宫密封 + 正压散热 | ✅ 已设计 |
| 强光/反光 | 偏振镜 + CLAHE预处理 | ✅ 软件层 |
| 高温（>40°C） | 散热鳍片 + 过温降频 | ✅ 已设计 |
| 湿度（>80%） | 全密封电池仓 + 硅胶干燥剂 | 🚧 验证中 |

**IP等级目标：** IP54（防尘防溅）

> 以上防护方案及状态来自 README.md。防护细节（密封材料、IP等级等）需工程验证。
