### 通信拓扑

```
[Jetson Nano] ──USB──> [摄像头]
      │
      ├──UART──> [电机驱动] ──> [左/右轮电机]
      │
      ├──I2C──> [IMU] + [里程计编码器]
      │
      ├──SPI──> [毫米波雷达]（可选）
      │
      └──GPIO──> [振动电机] + [抓取机构舵机]

[回充站] <──红外/无线充电──> [机器人]
      │
      └──WiFi──> [远程监控终端]
```

## 基础设施层

README.md 中描述的基础设施组件：

| 组件 | 说明 |
|------|------|
| ROS Master | ROS 中间件核心 |
| 参数服务器 | 运行时参数管理 |
| ROSbag录包 | 数据记录与回放 |
| 自动回充站 | 红外引导 + 无线充电 |
| 远程监控 | 基于 WiFi 的远程终端 |
| OTA升级 | 远程固件/软件更新 |
