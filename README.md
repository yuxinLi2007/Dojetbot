## 🚀 快速开始

> 🧪 *当前处于MVP阶段 —— 已在干沙区域完成测试*

```bash
# 克隆仓库
git clone https://github.com/your-org/IslandClean.git
cd IslandClean

# 安装依赖（ROS + 感知栈）
./scripts/install_deps.sh

# 编译工作空间
catkin_make
source devel/setup.bash

# 启动仿真（Gazebo + 干沙滩环境）
roslaunch island_clean sim_sand.launch

# 运行实体机器人（Jetson / 树莓派）
roslaunch island_clean robot_bringup.launch

# 启动垃圾检测节点
rosrun island_clean litter_detector.py

# 发送清理任务
roslaunch island_clean cleaning_mission.launch
```

## ⚙️ 硬件概述

### 行走机构

| 参数 | 数值 |
|------|------|
| 类型 | 履带式（V形自洁纹路） |
| 宽度 | 250mm |
| 接地长度 | 750mm |
| 悬挂 | 扭杆独立悬挂 |
| 材质 | 橡胶 + 凯夫拉帘线（防盐雾） |

**振动减阻模块（仅履带方案）：**

- 干沙区：50–80Hz，振幅1–2mm，阻力↓30–40%
- 湿沙区：自动关闭

> 🧪 **备选方案：** 轻量化低压轮胎（适用于硬质沙滩，成本更低）

### 收集机构

| 组件 | 规格 |
|------|------|
| 垃圾盒 | 3D打印，可拆卸 |
| 容积 | 2.5L |
| 目标垃圾 | 烟头、瓶盖、纸团、贝壳碎片、塑料碎片（<5cm） |

**抓取方式（方案验证中）：**

- 机械臂抓取（需设计自适应夹爪）
- 吸入式（适合轻质干垃圾）
- 铲斗式（适合大颗粒）

| 类别 | 示例 | 最小尺寸 |
|------|------|----------|

| 纸制品 | 纸巾、包装纸 | 随意 |

### 算法栈

```
输入帧
   │
   ▼
预处理（CLAHE去反光 + 沙尘滤波）
   │
   ▼
Grounding DINO / YOLOv8 推理
   │
   ▼
后处理（NMS + 置信度阈值0.5）
   │
   ▼
3D投影（深度估计 + 坐标转换）
   │
   ▼
抓取位姿发布 → 控制层
```

## 🔋 能源与自主性

### 电源系统

```
电池 (4S LiPo, 5000mAh)
   │
   ├─> 主控 (5V/3A) ──> Jetson + 摄像头 + 雷达
   │
   ├─> 电机驱动 (12V/10A) ──> 行走电机 ×2
   │
   ├─> 振动模块 (12V/5A) ──> 振动电机
   │
   └─> 抓取机构 (5V/2A) ──> 舵机/气泵
```

### 续航与充电

| 模式 | 功率 | 续航 |
|------|------|------|
| 干沙（无振动） | ~30W | 120min |
| 干沙（振动开启） | ~80W | 45min |
| 湿沙 | ~40W | 90min |
| 待机/回航 | ~10W | — |

**自动回充逻辑：**

1. 电量 < 25% → 中断清扫
2. 计算回航最短路径（A\*）
3. 红外引导对接充电站
4. 电量 > 90% → 返回中断点续扫

### 路径规划

- **全覆盖清扫：** 弓字形（Boustrophedon）+ 螺旋补扫边缘
- **动态重规划：** 电量 < 30% 自动切换回航路径
- **避障：** DWA（Dynamic Window Approach）
