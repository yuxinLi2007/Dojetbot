## 🌊 项目背景

### 为什么需要沙滩清洁机器人？

全球每年约 **800–1000 万吨** 塑料垃圾进入海洋，大量沉积在海滩。旅游旺季，重点海滩垃圾密度可达每平方公里数千件，远超生态警戒线。微塑料（粒径 < 5mm）已进入食物链，威胁海洋生态与人体健康。

> 仅惠州稺平半岛，2026 年已清理沙滩垃圾 **587 吨**，3091 处。

### 现有清洁方式的致命缺陷

| 清洁方式 | 问题 |
|----------|------|
| **人工清洁** | 成本高、效率低、工作环境恶劣，难以夜间作业 |
| **传统大型机械** | 噪音大、能耗高、破坏有机物层，无法进入小型海滩 |
| **突击式清理** | 缺乏长效机制，垃圾反复回潮 |

### 政策红利

| 政策 | 内容 |
|------|------|
| 海南省"海上环卫提质增效三年行动方案" | 2027 年底实现"美滩行动"品牌化、海岸线全覆盖 |
| 惠州稺平半岛整治 | 引入无人机巡查，大规模清理行动 |
| 蓝色循环模式（浙江台州） | 海洋塑料认证溢价 40%，"垃圾变黄金" |
| 禁塑令 + 湾长制 | 陆海统筹，源头管控 |
| 联合国塑料公约 | 2025 年签署，2026 年起陆续批准执行 |

## 🎯 产品定位

**核心定位**：小型智能沙滩清洁机器人（B2B / 文旅运营 / 政府采购）

**核心关键词**：`小型 AI` · `安静` · `夜间工作` · `高颜值` · `自动回充`

### 与传统方案对比

| 维度 | 传统方案 | 岛洁宝 |
|------|----------|--------|
| 体积 | 大型设备，难以进入小型海滩 | 小型化，适应各类海滩 |
| 作业时间 | 白天人工/机械，夜间停工 | 夜间静音工作，不打扰游客 |
| 智能化 | 机械筛分，无识别能力 | AI 识别烟头、微塑料等精细垃圾 |
| 能耗 | 柴油 / 高能耗 | 电动 + 自动回充 |
| 游客干扰 | 噪音大，影响体验 | 低噪音，动态避障 |

## 🔧 技术方案

### 2.1 行走机构

> MVP 阶段优先在**硬沙区域**测试验证

| 方案 | 规格 | 优势 | 劣势 |
|------|------|------|------|
| **履带式** | 宽 250mm，接地长 750mm，V 形自洁纹路，扭杆独立悬挂 | 沙地通过性强，贴合地形 | 结构复杂，成本较高 |
| **轻量化轮胎式** | 大直径低压轮胎 + 极轻车身 | 简单，低成本，易维护 | 载重有限 |

**振动减阻模块（履带方案）**：
- 干沙区自动开启高频低幅振动（50–80Hz，1–2mm 振幅）
- 行进阻力降低 30–40%，能耗增加约 50–80W
- 湿沙区自动关闭

### 2.2 感知与算法

**核心挑战**：沙滩环境恶劣（强光、反光、沙尘、水渍）

**技术方向**：
- 小样本 / 零样本垃圾识别（异常检测 + 视觉大模型，如 Grounding DINO）
- 传感器融合：视觉 + 毫米波雷达
- 数据增强提升鲁棒性

> ⚠️ **待解决**：沙滩覆盖层识别方案、视觉训练准确度与训练时长、算法具体实现路径

### 2.3 垃圾收集机构

**小型垃圾盒**（3D 打印）——可收集：
- 烟头、瓶盖、纸团、贝壳碎片、塑料碎片

**抓取方式**（方案探索中）：
- 机械臂抓取（需设计适应不同大小/干湿状态垃圾的抓夹形状）
- 吸入式或其他特殊结构

> ⚠️ **待解决**：不同垃圾形态的适应性

### 2.4 能源管理

**挑战**：沙滩面积大，有限电量需完成全区清扫

**技术方向**：
- 动态路径规划
- 根据实时电量调整清扫策略
- 自动回充站

## 💰 商业模式

参考**蓝色循环项目（浙江台州）**：前端精细分类 → 国际认证 → 海洋塑料溢价 40%，让"垃圾变黄金"。

**垃圾价值分层**：

| 类型 | 处理策略 | 说明 |
|------|----------|------|
| 高值垃圾（如干净塑料瓶） | 保值 / 增值 | 蓝色循环认证，2 倍以上溢价 |
| 低值垃圾 | 保量 / 兜底 | 政府补贴 + 生产者责任延伸制度 |
